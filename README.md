# DevTinderBE

Backend service for the DevTinder application, built with Node.js.

## Prerequisites

- Node.js (LTS recommended)
- npm (comes with Node.js)

## Setup

1. Install dependencies:
	 ```bash
	 npm install
	 ```
2. Create and configure any required environment variables (e.g. in a `.env` file if the project uses one).

## Running the project

- Start the development server:
	```bash
	npm run dev
	```

If `npm run dev` is not defined in `package.json`, you can start the app with:
```bash
node src/app.js
```

## Project structure

- `src/app.js` - main application entrypoint

## Scripts

Common npm scripts (check `package.json` for the exact list):

- `npm install` - install dependencies
- `npm run dev` - run the development server (if configured)
